## Obtaining HYWenHei-85W from Hanyi Fonts

1. Visit the official Hanyi Fonts website: https://www.hanyi.com.cn/
2. Navigate to **字体列表** (Font List) or **购买授权** (Purchase License)
3. For web use: Contact Hanyi for **Webfont** licensing at info@hanyi.com.cn or 010-68285561
4. Supported formats: `.woff2`, `.woff`, or `.ttf`
5. Name the file: `HYWenHei-85W.woff2` (or .woff / .ttf)
